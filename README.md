# Orient UBook Interactive Spec Site

Vercel-ready Next.js site that turns the Orient UBook MVP / Post-MVP roadmap into an interactive specification for investors and developers.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Notes

- Main source data lives in `src/data`.
- Public assets live in `public/assets`.
- Market figures are intentionally labelled as `official`, `deck assumption`, or `requires confirmation`.
- This is a spec-site only, not the real backend/admin MVP.
